Difference between HTML and HTML5

HTML HTML5
HTML is an older version of the markup language that has been used on the web for many years HTML5, on the other hand, introduced a significant number of new elements, attributes, APIs, and capabilities to modernize web development and enable richer web experiences
HTML has limited semantic elements for structuring content HTML5 introduced semantic elements like header, footer, nav, article, and section to provide better structure and meaning to web content
HTML forms have limited input types and attributes HTML5 introduced new input types (e.g., email, tel, date) and attributes for form validation and user input handling
